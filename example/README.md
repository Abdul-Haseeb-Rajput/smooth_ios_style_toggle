# Example app

Interactive tour of **smooth_ios_style_toggle** v0.2.0:

1. **setState** — `thumbWidth`, long labels, `semanticLabel`, `onToggle`
2. **SmoothToggleController** — `ValueListenableBuilder`
3. **Text inside thumb** — `textInsideThumb`
4. **Theme defaults** — `SmoothToggleTheme` only
5. **Large toggle** — max `trackHeight` (70), wide track, custom `thumbWidth`
6. **Compact** — min `trackHeight` (25)
7. **Disabled** — `enabled: false`
8. **No labels** — `showText: false`
9. **Thumb clamp** — narrow `trackWidth` (80)
10. **Keyboard** — `autofocus` + Space / Enter

```bash
flutter run
```
