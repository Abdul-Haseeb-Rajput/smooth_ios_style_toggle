# Security policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

1. Open a [private security advisory](https://github.com/Abdul-Haseeb-Rajput/smooth_ios_style_toggle/security/advisories/new), or  
2. Contact the maintainer via GitHub with minimal details and ask for a private channel.

Include:

- Description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Flutter / package version

We aim to acknowledge reports within **7 days** and will work on a fix or mitigation as soon as possible.

## Safe harbor

Good-faith security research that follows this policy will not be treated as a hostile act.
